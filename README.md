# Tienda Online 📱🛒

Aplicación móvil híbrida desarrollada con Ionic, Angular y Capacitor, que simula una tienda online con autenticación, consumo de API, carrito de compras y perfil dinámico. El proyecto incluye un backend en Node.js/Express con JWT para autenticación y un frontend en Ionic. Las funcionalidades principales abarcan la autenticación de usuarios mediante registro e inicio de sesión con JWT, el consumo de API REST para mostrar productos, un carrito de compras con almacenamiento local y sincronización de cantidades, un perfil dinámico que muestra nombre y correo del usuario autenticado, y la integración con Capacitor para compilar en Android/iOS. La estructura del proyecto se organiza en la carpeta `src/` con el código fuente del frontend Ionic/Angular, el archivo `server.js` con el backend en Node.js y endpoints de registro, login y productos, el archivo `package.json` con las dependencias del proyecto y la carpeta `assets/img/` con imágenes de productos y avatar de perfil. Para la instalación y ejecución se debe clonar el repositorio con `git clone https://github.com/TU_USUARIO/tienda-online.git` y entrar a la carpeta con `cd tienda-online`, instalar dependencias con `npm install`, ejecutar el backend con `node server.js` (servidor en http://localhost:3000) y ejecutar el frontend con `ionic serve` (aplicación en http://localhost:8100). Para la compilación con Capacitor se debe correr `ionic build`, luego `npx cap add android` y `npx cap add ios`, seguido de `npx cap copy`, y finalmente abrir el proyecto en Android Studio o Xcode con `npx cap open android` o `npx cap open ios`. En cuanto a pruebas realizadas, se ejecutaron en emulador Android Studio y en dispositivo físico Android, validando autenticación, carrito y perfil dinámico; la limitación es que no se publica en Play Store/App Store por falta de membresía.
